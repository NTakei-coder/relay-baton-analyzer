# Relay Baton Analyzer

4×100mリレーのバトンパス分析用Webアプリです。

## 今回の主な更新

- 分析情報を画面最上部へ移動
- 挙手コマを8番目、バトンパス完了コマを10番目の指示に変更
- バトンパス完了コマの定義を「バトンが受け手に渡り、渡し手の手が離れた瞬間」に変更
- 30mタイム・40mタイムを主要結果の先頭に大きく表示
- 30m/40mタイムの説明を「？」ボタンで表示
- 理論上の最速バトンタイムを表示
- 挙手時距離・パス完了時距離を主要結果に表示
- 挙手〜完了距離に基づく受け渡し評価を表示
- バトンパス完了0.1秒前の参考コマを表示し、前後に調整可能
- 出のタイミングの説明・評価・ゲージを追加
- 出のタイミングがぴったしだった場合の推定完了位置を表示
- 速度比較グラフの色分け、挙手位置・完了位置・交点ライン、受け渡し区間の塗りつぶしを追加
- グラフの軸ラベルと5m刻みの目盛りを追加
- PDF出力に対応

## ローカル確認

npm install

```

```

## Vercel公開

GitHubにこのフォルダの中身をアップロードし、Vercelでデプロイします。

推奨設定:

- Framework Preset: Vite
- Install Command: npm install
- Build Command: npm run build
- Output Directory: dist
- Node.js Version: 20.x

## 重要

GitHubへアップロードするときは、ZIPファイルそのものではなく、解凍した中身をアップロードしてください。GitHubのトップ画面で `package.json`, `index.html`, `src/main.jsx`, `src/App.jsx` が見えていれば正しい構成です。

動画はブラウザ内で読み込むだけで、サーバーへ保存しません。PDF出力はブラウザの印刷機能を使います。印刷画面で「PDFに保存」を選択してください。
